# 🎨 Identidade visual

## ✨ Conceito

A identidade visual do Zayro precisa comunicar **confiança, clareza e praticidade**.
O visual deve ser limpo, moderno e funcional — sem parecer frio ou genérico.

## 🧭 Direção de marca

| Atributo | Como deve parecer |
|---|---|
| Profissional | Passa seriedade e organização |
| Acolhedora | Não intimida o usuário |
| Objetiva | Mostra o próximo passo com clareza |
| Confiável | Reforça segurança e consistência |
| Mobile-first | Fácil de ler e tocar em telas pequenas |

## 🎨 Paleta sugerida

| Papel | Cor | Hex |
|---|---|---|
| Primary | Azul escuro | `#1E3A5F` |
| Primary Light | Azul médio | `#2E5A8F` |
| Secondary | Azul claro | `#4A90D9` |
| Accent | Celeste | `#87CEEB` |
| Surface | Branco | `#FFFFFF` |
| Neutral | Cinza médio | `#6B7280` |
| Error | Vermelho | `#DC2626` |

## 🔤 Tipografia

| Uso | Diretriz |
|---|---|
| Títulos | Destaque e peso forte |
| Subtítulos | Leves, claros e bem espaçados |
| Corpo | Leitura rápida e confortável |
| Botões | Texto curto, ação evidente |

## 🧱 Padrão de interface

- botões arredondados;
- cards com sombra leve;
- campos com borda suave;
- ícones simples e consistentes;
- bastante espaço em branco para leitura rápida.

## 🪄 Sensação desejada

O usuário deve bater o olho e entender:

1. onde está;
2. o que precisa fazer;
3. qual é a próxima ação.

## ✅ Resultado esperado

A interface deve parecer:

- séria, mas amigável;
- moderna, mas sem exageros;
- elegante, mas funcional.
