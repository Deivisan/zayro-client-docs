# Identidade visual

## Conceito

A identidade visual do Zayro precisa transmitir confiança, clareza e praticidade.
O visual deve ser limpo, moderno e funcional.

## Direção de marca

- profissional;
- acolhedora;
- objetiva;
- confiável;
- fácil de ler em mobile.

## Paleta sugerida

| Papel | Cor | Hex |
|---|---|---|
| Primary | Azul escuro | `#1E3A5F` |
| Primary Light | Azul médio | `#2E5A8F` |
| Secondary | Azul claro | `#4A90D9` |
| Accent | Celeste | `#87CEEB` |
| Surface | Branco | `#FFFFFF` |
| Neutral | Cinza médio | `#6B7280` |
| Error | Vermelho | `#DC2626` |

## Tipografia

- títulos com destaque e peso forte;
- textos curtos e legíveis;
- botões com hierarquia clara;
- uso consistente de tamanhos entre telas.

## Padrão de interface

- botões arredondados;
- cards com sombra leve;
- campos com borda suave;
- ícones simples e consistentes;
- bastante espaço em branco para leitura rápida.

## Objetivo visual

O usuário deve bater o olho e entender:

- onde está;
- o que precisa fazer;
- qual é a próxima ação.
