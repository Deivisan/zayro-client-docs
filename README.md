# Zayro — documentação pública do cliente

> Este repositório reúne, em linguagem clara e objetiva, a documentação pública do Zayro.

## O que é o Zayro

O Zayro é uma plataforma nacional que conecta candidatos à CNH a instrutores práticos de direção.

O foco do MVP é simples:

- apresentar a proposta;
- ajudar o candidato a informar sua situação;
- coletar cadastro e localização;
- listar instrutores disponíveis;
- permitir escolha de quantidade de aulas, datas e horários;
- exibir resumo, termos, pagamento e confirmação.
